				\item (4) Como $4 \in \mathbb{N}$, logo 7 é ímpar.
			\end{itemize}

		\end{block}

		\end{block}

	\begin{frame}{Questão 012}	
		\begin{block}{[Q012]}
			Logo abaixo há um argumento lógico para se dizer que não está chovendo agora.
			\begin{itemize}
				\item (1) Suponha que estivesse chovendo agora.
				\item (2) Se isto fosse verdade, o chão do estacionamento estaria molhado.
				\item (3) Mas o chão do estacionamento não está molhado.
				\item (4) Logo, não está chovendo agora.
			\end{itemize}
			Este argumento lógico utiliza qual tipo de prova?
		\end{block}
		\begin{enumerate}[label=(\Alph*)]
			\item Prova por construção
			\item Prova por contradição
			\item Prova por indução
			\item Não utiliza nenhum tipo de prova.
		\end{enumerate}
	\end{frame}

	\begin{frame}[shrink]{Questão 013}	
		\begin{block}{[Q013]}
			Logo abaixo há um argumento lógico para se dizer que todo circuito é 2-regular (todos os vértices têm grau 2).
			\begin{itemize}
				\item Para um circuito de 3 vértices, verifica-se que ele é 2-regular. Se um circuito de $n-1$ vértices $C_1$ for 2-regular, vamos mostrar que um circuito de $n$ vértices $C_2$ também o é. Seja $C_1$ o circuito $v_1 v_2 \ldots v_{n-1} v_1$. Vamos admitir que $C_1$ é 2-regular. É possível construir $C_2$ a partir de $C_1$: (i) adicionando um novo vértice $v_n$ vizinho de  $v_{n-1}$ e $v_1$ (logo $v_n$ tem grau 2), e (ii) removendo a aresta $v_{n-1}v_1$ (logo $v_{n-1}$ e $v_1$ voltam a ter grau 2). Desta forma, $C_2$ é 2-regular.
			\end{itemize}
			Este argumento lógico utiliza qual tipo de prova?
		\end{block}
		\begin{enumerate}[label=(\Alph*)]
			\item Prova por construção
			\item Prova por contradição
			\item Prova por indução
			\item Não utiliza nenhum tipo de prova.
		\end{enumerate}
	\end{frame}
